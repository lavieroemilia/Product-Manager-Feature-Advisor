# Decisiones

Registro de decisiones importantes: las que costaron algo (tiempo, alcance, recursos) o las que podrían haber ido de otra forma.

Cuando alguien pregunte "¿por qué se hizo así?", la respuesta está aquí — con el contexto que había en ese momento, no con la memoria de lo que pasó después.

---

## Cómo documentar una decisión

Nombre del archivo:
```
YYYY-MM-DD-nombre-corto.md
```
Ejemplo: `2024-11-03-descartar-exportacion.md`

---

## Plantilla

```markdown
# [Nombre de la decisión]

**Fecha:** YYYY-MM-DD  
**Feature relacionada:** [enlace si aplica]  
**Quién decidió:** [nombre o rol]

## Contexto
¿Qué situación llevó a tomar esta decisión?

## Opciones consideradas

| Opción | Pro | Contra |
|--------|-----|--------|
| Opción A | | |
| Opción B | | |

## Decisión
¿Qué se decidió y por qué?

## Consecuencias esperadas
¿Qué cambia a partir de esta decisión?

## Revisión (opcional)
¿Fue la decisión correcta? Completa esto semanas o meses después.
```
